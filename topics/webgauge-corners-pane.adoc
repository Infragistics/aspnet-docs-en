= Corners Pane

The Corners pane is used to determine the shape of the corners of your Linear gauge.

pick:[asp-net="link:{ApiPlatform}webui.ultrawebgauge{ApiVersion}~infragistics.ultragauge.resources.gauge~cornerextent.html[Extent]"]  -- Set this value to an integer from 0 to 100. The extent value determines the roundness of the corners of your Linear gauge.
