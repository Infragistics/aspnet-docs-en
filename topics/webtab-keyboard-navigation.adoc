= Keyboard Navigation

WebTab™ supports very easy navigation through the tabs using the keyboard.

* Left Arrow – Moves active tab to left.
* Right Arrow – Moves active tab to right.
* Home – Makes first tab active.
* End – Makes last tab active.
* Space/Enter – Selects active tab.

.Note:
[NOTE]
====
WebTab responds to the keyboard only if the control’s  pick:[asp-net="link:{ApiPlatform}web{ApiVersion}~infragistics.web.ui.layoutcontrols.webtab~enableactivation.html[EnableActivation]"]  property is set to True.
====
